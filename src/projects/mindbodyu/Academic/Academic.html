Academic Guide

Study Tips

  • Flashcards for memorization

  • Highlighting important areas of your notes

  • Writing out notes instead of typing them out
    allows you to process the information more

  • Watch Videos on the subject you are trying to learn

  • Join study groups! Trying to teach another person a topic
    will help your understand of the information!

Time Management

  • Utilizing Calendars and/or Planners to note when assignments are due and when exams and other events are

  • Establishing a daily routine/schedule

  • Setting aside time for non-academic activities so you can plan when to study

Stress Management

  • Staying Healthy - eating well, getting adequate sleep, and exercising regularly

  • Avoid procrastination - preventing assignments from piling up

  • Additional resources

Time Management Techniques

  • Pomodoro Technique - studying in 20-30 minute intervals with a 5 minute break , after doing this a few times take a 15-30 minute break

  • Eat the frog - select a large task to work on until complete, identify the next task when done

  • Time Boxing - set a schedule so that every time slot has a task designated to it. Once the time slot ends, proceed to next task

    Assignment Tracker

    How to Use the Assignment Tracker ?

    • Add Your Class: Locate the "Add Class" section at the top. Enter the name of the class in the "Class Name" field and choose a color to represent the class. Click the "Add Class" button to add this class to your list.
    • Add An Assignment: Move to the "Add Assignment" section. Start by typing the assignment name in the "Task Name" field. Choose the type of assignment (e.g., Quiz, Lab, Project) from the dropdown menu labeled "Assignment Type".
    • Set the Due Date: Pick the due date for the assignment using the "Due Date" date picker. This will help you track when each task is due.
    • Select the Class: Use the "Class Select" dropdown to choose which class the assignment belongs to. This links the task to the class you added previously.
    • Submit the Assignment: Click the "Add Assignment" button to add the assignment to your list. It will appear in the assignment table with details such as class, type, due date, and a checkbox to mark it as completed.
    • Manage Assignments: View and manage your assignments in the table. You can mark an assignment as completed by checking the box under "Completed". If you need to remove an assignment, click the "x" button in the "Remove Task" column.
